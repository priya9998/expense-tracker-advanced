#Expense Tracker Advanced

A React-based Expense Tracker application to manage income and expenses efficiently.

## Features
-Add Income
-Add Expense
-Edit Transaction 
-Delete Transaction
-Search Transactions
-Filter Transactions
-Category Management
-Statistics Dashboard
-Local Storage Support

## Technologies Used
-React
-JavaScript
-CSS
-Vite
